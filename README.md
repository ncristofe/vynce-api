
⚡ Vynce
✨ About

Vynce is a modern installable web application that delivers daily content through public APIs.

The app combines:

Weather information
Motivational quotes
Random content
Modern UI
Mobile-first experience

Built to practice frontend development with React.js and Vite.

🚀 Features
🌤️ Weather dashboard
💬 Daily quotes
👤 Random character cards
🖼️ Dynamic images
🌙 Dark mode
📱 Fully responsive
⚡ Fast loading
📦 Installable PWA
❤️ Favorites system
🛠️ Technologies
React.js
Vite
Tailwind CSS
Axios
React Router DOM
vite-plugin-pwa

src/
 ├── components/
 ├── pages/

 └── utils/
⚙️ Installation

Clone the repository:

git clone <repository-url>

Enter the project folder:

cd vynce

Install dependencies:

npm install

Start development server:

npm run dev
📱 PWA Support

Vynce can be installed on:

Android
Desktop
